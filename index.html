HTML5 FH Guide

Als begleitendes Projekt der LV Webtechnologien 1 wird ein HTML5 FH Guide umgesetzt, mit dem Ziel, die Raumbelegung direkt am Standort abrufen zu können

Die Anwendung ist für Mobile Devices optimiert und verwendet Bootstrap als CSS Framework, jQuery als JavaScript Library und AJAX zum Laden von Raumbelegungen von XML/JSON Files.
